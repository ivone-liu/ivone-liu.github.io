准备翻译《The Design of Web Apis》一书

Author: 刘一峰 Date: 2020-11-02  10:22 Category: 工作

最近简单浏览了一本书，书名叫做《The Design of Web Apis》。

虽然我们在日常软件开发中Api已经是一个用惯了的词，但是我们好似从来没有对这个词较真过，什么是Api，我问了几个同事，他们的回答基本上都是 “提供一个功能” 的意思，这本书给了我很大的启发。

从Api的设计思想拓展到软件的生命周期管理，这本书建议大家都读一下，但苦于没有找到合适的中文译本，我决定自己尝试翻译一下，如有翻译不当的地方可以给我邮件联系，还请多多包涵。

翻译的内容我会更新到这里，每天抽空就翻译一两段，翻译完可能要几个月时间。
